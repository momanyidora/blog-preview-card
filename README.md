What I learned
While building this project, I practiced creating responsive card layouts using Flexbox and improved my understanding of spacing, shadows, and hover states in CSS.
One thing I learned was how to center content perfectly using Flexbox:
body { display: flex; justify-content: center; align-items: center; min-height: 100vh;}
I also improved my understanding of hover effects:
.card:hover { box-shadow: 10px 10px 0 black; cursor: pointer;}
